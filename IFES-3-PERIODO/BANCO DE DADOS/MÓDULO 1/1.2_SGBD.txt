DEFINIÇÃO DE SGBD
    Software que incorpora funções de "definição, recuperação e alteração" de dados em um banco de dados.
    - Sistema de Gerenciamente de Banco de Dados (SGBD)

Exemplos de sistemas que podem acessar um mesmo banco de dados:
     - Recursos Humanos
     - Contabilidade
     - Estoque
     - Marketing
     - Projetos


CARACTERÍSTICAS IMPOSTANTES DE UM SGBD:
    - Natureza descritiva do SGBD
    - Isolamento ente programas e dados
    - Abstração de dados
    - Multiplas visões de dados:
        - Independência de dados
        - Segurança
    - Transaçõe smulti-usuparuos e controle de concorrêncoa
        - ACID:
            - Atomicidade
            - Consistência
            - Isolamento
            - Durabilidade
