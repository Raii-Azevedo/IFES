ORIENTAÇÕES DO TRABALHO

Desenvolva um algoritmo em JAVA que permita a entrada do nome de um aluno e as notas de três provas, sendo que cada nota deve estar no intervalo de 0 a 10 pontos. O sistema deverá verificar se as notas são válidas, informando ao usuário quando uma nota for digitada fora desse intervalo e solicitando uma nova entrada.

Após a entrada das notas, o algoritmo deve calcular a soma das três notas. Em seguida, deverá determinar se o aluno foi aprovado ou reprovado com base na seguinte regra: se a soma das notas for igual ou maior que 18, o aluno estará aprovado; caso contrário, estará reprovado. Se a média das notas estiver entre 15 pontos e 17.9, o aluno fará a recuperação final.

Valor do trabalho: 10,0 pontos.

Observação: O trabalho poderá ser realizado em dupla.
